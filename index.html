MySQL创建用户与授权

创建用户

CREATE USER ‘username’@’host’ IDENTIFIED BY ‘password’;
说明：

  • username：你将创建的用户名
  • host：指定该用户在哪个主机上可以登陆，如果是本地用户可用localhost，如果想让该用户可以从任意远程主机登陆，可以使用通配符%
  • password：该用户的登陆密码，密码可以为空，如果为空则该用户可以不需要密码登陆服务器
    实例：

    CREATE USER ‘bookkeeping’@’%’ IDENTIFIED BY ‘***‘;

    授权

    GRANT privileges ON databasename.tablename TO ‘username’@’host’;

    说明:
  • privileges：用户的操作权限，如SELECT，INSERT，UPDATE等，如果要授予所的权限则使用ALL
  • databasename：数据库名
  • tablename：表名，如果要授予该用户对所有数据库和表的相应操作权限则可用表示，如.*
    实力：

    GRANT ALL ON bookkeeping.* TO ‘bookkeeping’@’%’;

PS:参考资料：https://www.cnblogs.com/sos-blue/p/6852945.html

linux 安装nexus3

由于 https://www.sonatype.com/download-oss-sonatype 下载一直被拒绝。所以将采用 docker image的方式安装 nexus3

  1. 拉取image
  2. 启动 image
  3. 配置 nexus

  • 有一个环境变量用于将JVM参数传递给启动脚本

    • INSTALL4J_ADD_VM_PARAMS，传递给Install4J启动脚本。默认为-Xms1200m -Xmx1200m -XX:MaxDirectMemorySize=2g -Djava.util.prefs.userRoot=${NEXUS_DATA}/javaprefs。

      可以在运行时进行调整：

      $ docker run -d -p 8081:8081 –name nexus -e INSTALL4J_ADD_VM_PARAMS=”-Xms2g -Xmx2g -XX:MaxDirectMemorySize=3g -Djava.util.prefs.userRoot=/some-other-dir” sonatype/nexus3

      尤其要注意的是，-Djava.util.prefs.userRoot=/some-other-dir可以将其设置为永久路径，如果重新启动容器，它将保持已安装的Nexus Repository License。

  • 另一个环境变量可用于控制Nexus上下文路径

    • NEXUS_CONTEXT，默认为/

      可以在运行时提供：

      $ docker run -d -p 8081:8081 –name nexus -e NEXUS_CONTEXT=nexus sonatype/nexus3

持续数据

有两种使用Docker处理持久性存储需求的通用方法。有关 其他信息，请参见在容器中管理数据。

  1. 使用泊坞窗卷。由于Docker卷是永久性的，因此可以专门为此创建一个卷。这是推荐的方法。
    $ docker volume create –name nexus-data
    $ docker run -d -p 8081:8081 –name nexus -v nexus-data:/nexus-data sonatype/nexus3
  2. 将主机目录挂载为卷。这不是可移植的，因为它依赖于在主机上具有正确权限的现有目录。但是，在需要将该卷分配给某些特定基础存储的某些情况下，此方法很有用。
    $ mkdir /some/dir/nexus-data && chown -R 200 /some/dir/nexus-data
    $ docker run -d -p 8081:8081 –name nexus -v /some/dir/nexus-data:/nexus-data sonatype/nexus3

我使用的启动配置：

docker run -d -p 30001:8081 –name nexus -v /usr/local/nexus:/nexus-data sonatype/nexus3

PS:参考资源：https://github.com/sonatype/docker-nexus3
